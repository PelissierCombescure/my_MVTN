Sure! Here's a list of commonly used shortcuts and commands for `screen`:

### Basic Screen Commands

| Shortcut/Command                   | Action                                              |
|------------------------------------|-----------------------------------------------------|
| `screen`                           | Start a new screen session                          |
| `screen -S session_name`           | Start a new screen session with a specified name    |
| `screen -ls`                       | List all active screen sessions                     |
| `screen -r session_name`           | Reconnect to a specific detached session            |
| `screen -r session_id`             | Reconnect to a session using its ID                 |
| `screen -d -r session_name`        | Detach another user's screen and reattach it        |
| `exit`                             | Close the current screen session                    |
| `screen -S <session id> -X quit`   | Deelete session using its ID                        |
### Detaching and Reattaching

| Shortcut                            | Action                                              |
|-------------------------------------|-----------------------------------------------------|
| `Ctrl + A, D`                       | Detach from the current screen session              |
| `Ctrl + A, ` (then release)         | Enter command mode (allows you to enter commands)   |
| `Ctrl + A, R`                       | Detach and reattach the current session             |

### Creating and Managing Windows

| Shortcut                            | Action                                              |
|-------------------------------------|-----------------------------------------------------|
| `Ctrl + A, C`                       | Create a new window in the current screen session   |
| `Ctrl + A, N`                       | Switch to the next window                           |
| `Ctrl + A, P`                       | Switch to the previous window                       |
| `Ctrl + A, "`                       | List all windows (allows switching between them)    |
| `Ctrl + A, K`                       | Kill the current window                             |
| `Ctrl + A, 0-9`                     | Switch to the window with the specified number      |

### Scrolling and Copying

| Shortcut                            | Action                                             |
|-------------------------------------|----------------------------------------------------|
| `Ctrl + A, Esc`                     | Enter copy mode (scroll back)                      |
| Arrow keys                          | Scroll up/down in copy mode                        |
| `Space`                             | Start selecting text in copy mode                  |
| `Enter`                             | Copy the selected text to the buffer               |
| `Ctrl + A, ]`                       | Paste the copied text                              |

### Miscellaneous

| Shortcut                            | Action                                              |
|-------------------------------------|-----------------------------------------------------|
| `Ctrl + A, ?`                       | Show help (list all commands and shortcuts)         |
| `Ctrl + A, X`                       | Lock the screen                                     |
| `Ctrl + A, T`                       | Show the current time in the status line            |

### Example of Using Shortcuts

1. Start a screen session: `screen -S mysession`
2. Run a command.
3. Detach: `Ctrl + A, D`
4. Reconnect: `screen -r mysession`
5. Create a new window: `Ctrl + A, C`
6. Switch to the next window: `Ctrl + A, N`
7. Scroll back: `Ctrl + A, Esc`, then use arrow keys.
8. Exit the session: `exit` (in all windows to fully close).

This list should cover most of the common operations you'll use with `screen`. Let me know if you have any questions or need more details!
